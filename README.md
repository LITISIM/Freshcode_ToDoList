# ToDo List App (Fullstack)

Це навчальний проєкт списку завдань, реалізований як Fullstack-додаток. Дозволяє додавати, видаляти та відмічати завдання як виконані.

## Особливості
- **Frontend**: Чистий JavaScript (Vanilla JS), CSS3 (адаптивна верстка в rem/vw).
- **Backend**: Node.js та фреймворк Express.
- **Збереження даних**: Дані зберігаються у файлі `tasks.json` на сервері.

## Структура проєкту
- `index.html` — основна розмітка додатка.
- `style.css` — стилізація інтерфейсу.
- `script.js` — клієнтська логіка та взаємодія з API.
- `server.js` — серверна частина (Node.js/Express).
- `tasks.json` — база даних проєкту (створюється автоматично).
